## 1. 導入手順

### 必要なもの

- パソコン（Mac / Windows）
- Node.js v18以上（[nodejs.org](https://nodejs.org/) からダウンロード）
- Anthropic APIキー（[console.anthropic.com](https://console.anthropic.com/) で取得）

### インストール手順

1. 配布された `contentgift.zip` を解凍
2. 解凍フォルダに `.env.local` ファイルを作成: `ANTHROPIC_API_KEY=sk-ant-あなたのAPIキー`
3. ターミナルで `npm install` → `npm run dev`
4. ブラウザで表示されたURL（通常 http://localhost:3906 ）を開く

### アプリ内マニュアル

ヘッダー右上の「📖 使い方」ボタンからアプリ内マニュアルも確認できます。

## 2. 全体の流れ

```
⚙️ 設定 → 💡 アイデア → 📄 コンテンツ → ✅ 品質チェック → 📥 PDF保存
```

1. **⚙️ 設定**: 商品情報・ターゲット・用途を入力
2. **💡 アイデア**: AIが6つの特典アイデアをスコア付きで提案
3. **📄 コンテンツ**: 選んだアイデアの本文をAIが自動生成
4. **✅ 品質チェック**: 定量・定性の両面からAIが評価し改善提案
5. **📥 PDF保存**: ブラウザの印刷機能でPDF化

## 3. 商品情報を設定する

### 特に重要な設定項目

| 項目 | なぜ重要か |
|------|-----------|
| 特典の用途 | リスト獲得用 / セミナー特典でアイデアの方向性が変わる |
| ターゲットの悩み | 具体的なほど「刺さる」特典タイトルになる |
| 競合の特典 | 差別化されたアイデアを出すための材料 |
| 希望形式 | チェックリスト・動画台本等。AIにおまかせも可能 |

### 対応する6種類の特典形式

| 形式 | 特徴 | 向いているケース |
|------|------|----------------|
| ✅ チェックリスト | すぐ使える・行動しやすい | 「やることがわからない」人向け |
| 📖 ガイドブック | ステップバイステップで学べる | 体系的な知識提供 |
| 🎬 動画台本 | 話し言葉で演出指示付き | 動画コンテンツを作りたい人 |
| 📝 ワークシート | 書き込み式で自己分析 | 自分ごと化させたい場合 |
| 📋 テンプレート集 | コピペで使えるひな形 | 「時短」を求めている人向け |
| 💎 スワイプファイル | 成功事例集 | 参考にしたい人向け |

## 4. アイデアを生成する

「💡 アイデアを生成」ボタンで、AIが6つの特典アイデアを提案します。

### 6つの評価指標

| 指標 | 説明 |
|------|------|
| リスト獲得力 | 「今すぐ欲しい」と思わせる力 |
| 感動指数 | 「無料でこのレベル？」と思わせる力 |
| 架け橋度 | 有料商品への自然な導線の強さ |
| 即実践度 | 受け取ってすぐ使える具体性 |
| セミナー参加率UP | セミナーへの期待を高める力 |
| 制作しやすさ | 実際に制作する難易度 |

### アイデアカードの見方

| セクション | 内容 |
|-----------|------|
| タイトル | 見込み客に刺さるコピー |
| 形式 | チェックリスト/ガイドブック等 |
| フック | LP・SNSで使えるキャッチコピー |
| なぜ効くか | 心理的な理由 |
| 架け橋戦略 | 有料商品へのつなげ方 |
| 構成案 | セクションの概要 |

### タイトル案の生成

各アイデアカードの「🎯 タイトル案を5つ生成」ボタンで、5種類の切り口のタイトルが提案されます。

| パターン | 例 |
|---------|-----|
| 数字×結果型 | 「7日間で月収100万円を達成する5つのステップ」 |
| ターゲット共感型 | 「SNS集客に疲れた起業家のための新しい集客術」 |
| 好奇心型 | 「トップ3%のプロモーターだけが知る提案の型」 |
| 即効ツール型 | 「コピペで完成！クライアント獲得メールテンプレート10選」 |
| Before/After型 | 「時給2,000円→月200万円を実現した戦略の全記録」 |

気に入ったタイトルをクリックすると、そのアイデアのタイトルが差し替わります。

## 5. コンテンツを生成する

アイデアカードの「📄 このアイデアでコンテンツを生成」で本文を自動生成。

### 生成されるコンテンツ構成

1. タイトル & サブタイトル
2. はじめに（500〜800文字）
3. 本編セクション（8〜12セクション、各600〜1,500文字）
4. おわりに（300〜500文字）
5. CTA — 有料商品への自然な導線（200〜400文字）

### コピー機能（3種類）

| コピー方法 | 用途 |
|-----------|------|
| 📋 全文コピー | テキスト全体をそのままコピー |
| 📝 Markdown形式コピー | 見出し・リスト付きの構造化テキスト。ブログやメモアプリに |
| 🎨 デザインツール用コピー | スライド構成（ページ区切り付き）。Canva等のデザインツールに流し込む用 |

### デザインツールでの仕上げ方

1. 「🎨 デザインツール用コピー」でコピー
2. Canva等でA4縦テンプレートを開く
3. 各スライドにコピーしたテキストを配置
4. 図解・アイコン・デザインを追加して仕上げ

## 6. 品質チェック

「✅ 品質チェック」ボタンでAIが評価。

### 一括品質チェック

複数コンテンツがある場合、「🔄 すべてのコンテンツを一括チェック」で順番に自動分析。
進捗バーで処理状況を確認でき、完了後にスコア一覧が表示されます。

### 定量評価（5項目）

| 項目 | 目標値 |
|------|--------|
| 文字数 | 8,000〜15,000 |
| セクション数 | 8〜12 |
| アクションアイテム | 20+ |
| 具体例 | 10+ |
| データポイント | 5+ |

### 定性評価（6項目、各10点）

明瞭さ / 即実践性 / 独自性 / 感情インパクト / 知覚価値 / 架け橋効果

### 判定基準

| 判定 | スコア | 意味 |
|------|--------|------|
| Excellent | 85+ | そのまま使えるレベル |
| Good | 65〜84 | 微調整で使えるレベル |
| Needs Improvement | 〜64 | 改善が必要 |

### 書き換え提案（自動適用対応）

各提案に「📍 該当箇所」が表示されるので、どのセクションの改善かがわかります。

- 「✅ この書き換えを適用」で1件ずつ適用
- 「🔄 すべての提案を一括適用」で全提案を一度に反映

適用後は自動でコンテンツが更新・保存されます。

## 7. コンテンツの出力方法

### 出力方法の使い分け

| コンテンツ種別 | 推奨出力方法 | 理由 |
|-------------|------------|------|
| 動画台本 | PDF保存（ブラウザ印刷） | デザインより読みやすさ重視 |
| チェックリスト | Canvaで仕上げ | チェックボックスやアイコンが映える |
| ガイドブック | Canvaで仕上げ | 図解・レイアウトが価値を上げる |
| ワークシート | Canvaで仕上げ | 書き込み欄のデザインが重要 |
| テンプレート集 | Canvaで仕上げ | 見た目がそのまま「使えるツール」感に |
| スワイプファイル | Canvaで仕上げ | 事例の見せ方でプロ感が出る |

### 動画台本 → PDF保存

1. コンテンツタブで「📥 PDF保存」をクリック
2. 印刷プレビューが開く
3. 「送信先」を「PDFとして保存」に変更
4. 「保存」をクリック

> Chromeの場合、「詳細設定」→「背景のグラフィック」にチェックを入れると色が反映されます。

### 資料系 → Canvaで仕上げ（推奨）

資料系の特典はデザイン性（図解・アイコン・レイアウト）が価値に直結するため、**Canvaでの仕上げを強く推奨**します。

## 7.5. Canvaで特典PDFを仕上げる方法

Content Giftで生成したテキストを、Canvaでプロ品質のPDFに仕上げる手順です。

### STEP 1: テンプレートを選ぶ

1. Canva（canva.com）にログイン
2. 「デザインを作成」→「A4文書」を選択（縦型：210×297mm）
3. 左メニュー「テンプレート」から好みのデザインを検索
4. ブランドカラーに近い配色のテンプレートを選ぶと効率的

#### おすすめ検索キーワード

| 特典の形式 | Canva検索キーワード |
|-----------|-------------------|
| チェックリスト | 「checklist」「to-do list」 |
| ガイドブック | 「ebook」「guide」「whitepaper」 |
| ワークシート | 「worksheet」「workbook」「planner」 |
| テンプレート集 | 「template」「swipe file」「resource」 |
| スワイプファイル | 「case study」「portfolio」「lookbook」 |

### STEP 2: テキストを流し込む

1. Content Giftのコンテンツタブで「🎨 デザインツール用コピー」をクリック
2. コピーされたテキストは「=== ページ 1: 表紙 ===」のようにページ区切りされています
3. Canvaの各ページに対応するテキストを貼り付け
4. 表紙ページ → タイトル・サブタイトルを大きく配置
5. 本文ページ → セクション見出し + 本文を配置

> 💡 テキスト量が多い場合はページを分割してOK。Canvaで「ページを追加」は簡単です。

### STEP 3: デザイン要素を追加

1. **アイコン**: 左メニュー「素材」→「アイコン」で検索。各セクションにアイコンを添えると視認性UP
2. **図形**: 見出しの背景に帯（長方形）を敷く、重要項目を枠で囲む
3. **チェックボックス**: チェックリスト形式なら□アイコンを各項目に配置
4. **区切り線**: セクション間に線やデザイン要素を入れて読みやすく
5. **吹き出し・ボックス**: 重要な数字やTIPSを目立たせる

### STEP 4: ブランディングを統一

1. Content Giftで設定したブランドカラー（メインカラー・アクセントカラー）をCanvaでも使用
2. フォントは**見出し用**（太字・インパクト）と**本文用**（読みやすさ重視）の2種類に統一
3. ロゴがあれば全ページのヘッダーまたはフッターに小さく配置
4. フッターにページ番号を追加

> 💡 Canva Pro版なら「ブランドキット」でカラー・フォントを保存でき、複数クライアント案件で効率的に使い回せます。

### STEP 5: PDFで書き出す

1. 右上「共有」→「ダウンロード」をクリック
2. ファイルの種類で「PDF（印刷）」を選択（高品質）
3. 「ダウンロード」をクリック

### デザインの注意点

- **文字を詰めすぎない** — 余白を十分に取る（上下左右20mm以上のマージン推奨）
- **1ページに情報を詰めすぎない** — 見出し1つ + 本文で1ページが読みやすい
- **色を使いすぎない** — メインカラー・アクセントカラー + 黒 + 白の4色で統一感を出す
- **フォントサイズ** — 見出し24pt以上、本文12pt以上が推奨

## 8. バックアップと復元

### バックアップ
トップページ右上の「バックアップ」ボタンで全データをJSONファイルに保存。

### 復元
「復元」ボタンでバックアップファイルを読み込み。同じIDは上書き、新しいものは追加。

### 推奨タイミング
- 週に1回
- 大量のコンテンツ生成後
- ブラウザのキャッシュクリア前
- 別のPCで使い始めるとき

## 9. クライアント案件での使い方

### プロモーターとしての活用フロー

**STEP 1: ヒアリング**
1. クライアントの商品情報を入力
2. ターゲットの悩み・理想を詳しく設定
3. 競合がどんな特典を出しているか調査・入力

**STEP 2: アイデア提案**
1. AIが6つのアイデアを生成
2. スコアと架け橋戦略をもとにクライアントと相談
3. 最適なアイデアを選択

**STEP 3: コンテンツ制作**
1. 選んだアイデアのコンテンツを生成
2. 品質チェックで客観評価
3. 書き換え提案を反映
4. PDF保存してクライアントに納品

## 10. 効果的な特典の作り方

### リスト獲得力を最大化する3つのコツ

1. **「知識」より「ツール」を提供する** — チェックリスト・テンプレートは行動のハードルが低い
2. **タイトルに具体的な数字を入れる** — 「7つのステップ」「30日で」
3. **競合がやっていない切り口を選ぶ** — 同じジャンルでも角度を変えて差別化

### 架け橋効果を高めるコツ

特典で「一部を深く」出す。全体の概要を浅く出すより、1つのテーマを深く掘り下げると「他のテーマも知りたい」と思わせることができる。

### やってはいけないこと

- 有料商品の内容を全部出す → 「もう買わなくていい」と思われる
- 抽象的・一般的な内容だけ → 「ググれば出てくる」と思われる
- 押し売り感のあるCTA → 信頼が崩れる

## 11. よくある質問

### Q: APIキーの料金は？
A: Anthropic APIは従量課金。アイデア生成1回約$0.05〜0.10、コンテンツ生成1回約$0.15〜0.30（ボリューム増加のため）、品質チェック1回約$0.05、タイトル生成1回約$0.03。月に数十回使っても数ドル程度。

### Q: データはどこに保存される？
A: ブラウザのlocalStorage。サーバーには保存されません。定期的にバックアップしてください。

### Q: 生成されたコンテンツはそのまま使える？
A: 品質チェックで「Excellent」判定なら基本そのまま使えます。ただし、あなたの言葉に合わせた微調整を推奨します。

### Q: 動画台本はそのまま読めば動画が作れる？
A: はい。話し言葉で書かれており、（間を置く）（画面に表示）等の演出指示付きなので、そのまま読み上げて撮影できます。

### Q: 別のPCでも使える？
A: バックアップ→復元でデータ移行できます。
